# 项目结构说明

## 📁 文件结构
```
src/
├── App.vue          # 主应用组件
├── main.js          # 应用入口文件
├── router/
│   └── index.js     # 路由配置
├── components/      # 公共组件
├── view/           # 页面组件
└── assets/         # 静态资源
```

## 🚀 启动项目
```bash
npm install    # 安装依赖
npm run dev    # 启动开发服务器
npm run build  # 构建生产版本
```

## 🔧 主要技术栈
- Vue 3 - 前端框架
- Vue Router - 路由管理
- Vite - 构建工具
- Bootstrap - UI 框架
- jQuery - DOM 操作
- Animate.css - 动画效果
